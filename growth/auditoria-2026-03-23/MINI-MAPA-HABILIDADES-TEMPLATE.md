# Mini-Mapa de Habilidades — Template para Entrega Manual

**Propósito:** Lead magnet entregable HOY sin MVP. Análisis manual con Claude → PDF de 1 página → entrega en <24h.

---

## INSTRUCCIONES PARA D (Operador)

### Input necesario del usuario:
1. Sector/industria actual
2. Años de experiencia
3. Rol actual o último
4. 2-3 logros de los que esté orgulloso
5. (Opcional) CV adjunto

### Proceso:
1. Usuario responde encuesta o manda email con info
2. Tú procesas con Claude usando el prompt de abajo
3. Generas el PDF con el template de abajo
4. Envías por email en <24h

### Prompt para Claude:

```
Analiza el siguiente perfil profesional y extrae:

1. 10-15 habilidades clave (mezcla de hard y soft skills)
2. Para cada habilidad: nivel estimado (Básico/Intermedio/Avanzado)
3. 3-5 roles del mercado laboral donde estas habilidades son valiosas
4. 1 "habilidad oculta" que probablemente tiene y no está vendiendo

Perfil:
- Sector: [X]
- Años experiencia: [X]
- Rol actual: [X]
- Logros: [X]
- CV: [si lo tiene]

Output en formato estructurado para copiar a un PDF.
Tono: profesional pero cercano, sin jerga corporativa.
Enfócate en habilidades transferibles, no en títulos o certificaciones.
```
